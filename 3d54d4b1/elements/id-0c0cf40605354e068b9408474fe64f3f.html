Route 53 ()
Definition: Amazon Route 53 is a highly available and scalable Domain Name System (DNS) web service. You can use Route 53 to perform three main functions in any combination: domain registration, DNS routing, and health checking. Source: Amazon Source reference: https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/Welcome.html Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Route 53 can manage DNS records for AWS services and provide DNS for external resources as well. eira:iopDimension: Structural IoP LOST view: TVI-Networking Infrastructure Enablers [Traceability] eira:iopSaliency: Domain Name Service ABB is salient for semantic interoperability because it implements a catalogue of internet addresses associated with the corresponding names to facilitate accessibility to public administration services.
