MongoDB ()
Description: MongoDB is a source-available, cross-platform, document-oriented database program. Classified as a NoSQL database product, MongoDB utilizes JSON-like documents with optional schemas. MongoDB is developed by MongoDB Inc. and current versions are licensed under the Server Side Public License (SSPL). MongoDB is a member of the MACH Alliance. Example: An example application of MongoDB is used to store the data transmited to central VIES in JSON. MongoDB is a JSON Native non-sql Database which optimizes the operations using this standard.
