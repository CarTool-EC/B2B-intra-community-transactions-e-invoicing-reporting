Velero Backup/Restore Image ()
Definition: Velero is an open source tool to safely backup and restore, perform disaster recovery, and migrate Kubernetes cluster resources and persistent volumes. Extra Information: Velero (formerly Heptio Ark) gives you tools to back up and restore your Kubernetes cluster resources and persistent volumes. You can run Velero with a cloud provider or on-premises. Velero lets you: - Take backups of your cluster and restore in case of loss. - Migrate cluster resources to other clusters. - Replicate your production cluster to development and testing clusters. Velero consists of: - A server that runs on your cluster - A command-line client that runs locally Source: Velero Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Central VIES uses a managed kubernetes service as container orchestrator. Containers are deployed on top of kubernetes and some of them requires persistens storage, such as MongoDB database. In order to run backup and restore, as part of the disaster recovery strategy, Velero provides the funcionalities to backup the services which are running on top of Kubernetes. For other services not running on top of Kubernetes cluster (such as message broker), backup and restore will have to be implemented following vendor's recomendations. eira:iopDimension: Structural IoP LOST view: TVI-Central VIES Deployment Model eira:iopSaliency: Container ABB is salient for technical interoperability because it provides the packaged software that contains or nurtures the functionalities of public services.
