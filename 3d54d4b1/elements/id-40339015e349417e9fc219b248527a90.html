Kong API Gateway ()
Definition: Kong Gateway is the world’s most adopted API gateway. An API gateway is a reverse proxy that lets you manage, configure, and route requests to your APIs. The Kong Gateway runs in front of any RESTful API and can be extended through modules and plugins.
