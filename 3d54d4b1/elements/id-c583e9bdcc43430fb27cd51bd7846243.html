PaaS ()
Definition: PaaS ABB is a Technology Service that enables the procedures for delivering application infrastructure (middleware) capabilities as a service. There are multiple types of PaaS (xPaaS), including, among many more, application platform as a service (aPaaS), integration PaaS (iPaaS), API management PaaS (apimPaaS), function PaaS (fPaaS), business analytics PaaS (baPaaS), IoT PaaS and database PaaS (dbPaaS). PaaS capability can be delivered as provider-managed or self-managed, multitenant or dedicated. Source: Gartner Source reference: https://www.gartner.com/en/information-technology/glossary/platform-as-a-service-paas Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): OpenSHIFT is a secure and an enterprise-grade cloud-based container orchestration system with Kubernetes at the backend.
