Route Table ()
Definition: A route table contains a set of rules, called routes, that determine where network traffic from your subnet or gateway is directed. Source: Amazon Source reference: https://docs.aws.amazon.com/vpc/latest/userguide/VPC_Route_Tables.html Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Main route table in AWS. The route table that automatically comes with your VPC.
