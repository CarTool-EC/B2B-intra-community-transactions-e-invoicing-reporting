Quartz Scheduler ()
Description: Quartz is an open source job-scheduling framework written entirely in Java and designed for use in both J2SE and J2EE applications. It offers great flexibility without sacrificing simplicity. You can create complex schedules for executing any job. Examples are e.g. tasks that run daily, every other Friday at 7:30 p.m. or only on the last day of every month. Example: An example application of Quartz Scheduler is when sending einvoices on batches. Administrator can configure a cron expression for transferring einvoices to Central VIES once a day. If the solution is deployed using kubernetes, quartz will be not necessary since the developer/architect can take advantaga of the Kubernetes Scheduled Jobs.
