IaaS ()
Definition: IaaS ABB is a Technology Service that enables the procedures to implement a standardized, highly automated offering in which computing resources owned by a service provider, complemented by storage and networking capabilities, are offered to customers on demand. Resources are scalable and elastic in near real-time and metered by use. Self-service interfaces, including an API and a graphical user interface (GUI), are exposed directly to customers. Resources may be single-tenant or multitenant and are hosted by the service provider or on-premises in a customer’s data centre. Source: Gartner Source reference: https://www.gartner.com/en/information-technology/glossary/infrastructure-as-a-service-iaas Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): EBSI, the European Blockchain Service Infrastructure. lockchain Services Infrastructure (EBSI) aims to leverage the power of blockchain for the public good.
